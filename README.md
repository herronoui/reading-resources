# Reading Resources

A simple list of books worth reading for robotics, controls, and aerospace.

**Live site:** [herronoui.github.io/reading-resources](https://herronoui.github.io/reading-resources)

## Run locally

```bash
open index.html
```

No build step. Just HTML.

## Add a book

Edit `index.html` and add a `<li>` under the right section.
